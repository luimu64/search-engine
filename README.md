# Zap
Small and zappy search aggregator using different programming related public APIs.<br><br>
![image](https://github.com/luimu64/zap/assets/54173370/d629ee8f-d763-4a50-a69d-57bda65c27cf)

## Running
`git clone https://github.com/luimu64/zap`\
`cd zap && bun i`\
`bun dev`
